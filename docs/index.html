Everything You Need

Comprehensive DICOM functionality in one beautiful application

Fast Image Viewing

Lightning-fast DICOM rendering with interactive window/level adjustment. Decode times under 50ms.

Tag Editor

View, edit, and delete DICOM tags with validation. Search, filter, and manage metadata effortlessly.

HIPAA Anonymization

Compliant de-identification with Basic, Full, and Research templates. Protect patient privacy.

Multi-Series Navigation

Browse studies with multiple series and instances. Navigate complex imaging studies effortlessly.

DICOMweb Client

Modern PACS connectivity with QIDO-RS and WADO-RS. Query and retrieve from cloud PACS systems.

Export & Convert

Export images as PNG with custom windowing. Export tags as JSON or XML for analysis.
